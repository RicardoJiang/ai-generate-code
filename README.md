# ai-generate-code

AI 生成的各类可视化网页项目集合。

## 项目列表

### 3D 圣诞树
[https://ricardojiang.github.io/ai-generate-code/christmas-tree/dist/index.html](https://ricardojiang.github.io/ai-generate-code/christmas-tree/dist/index.html)

基于 Three.js 的 3D 交互式圣诞树，带有动画效果和节日氛围。

### Now in Kotlin - 设计展示页
[https://ricardojiang.github.io/ai-generate-code/now-in-kotlin-design/index.html](https://ricardojiang.github.io/ai-generate-code/now-in-kotlin-design/index.html)

Kotlin 编程语言的高保真设计展示页面，采用 Tailwind CSS 实现现代化 UI，包含代码高亮、动画效果和响应式设计。

### 黄金定投收益分析
[https://ricardojiang.github.io/ai-generate-code/gold/index.html](https://ricardojiang.github.io/ai-generate-code/gold/index.html)

黄金定投 14 年（2012-2025）复盘分析报告，展示每年投资收益率和累计收益情况。使用 Chart.js 进行数据可视化，采用 Modified Dietz 方法计算年度收益率。

### 投资策略对比 - 全天候 vs 永久组合
[https://ricardojiang.github.io/ai-generate-code/harry/index.html](https://ricardojiang.github.io/ai-generate-code/harry/index.html)

2025 年投资策略深度对决，对比全天候策略（All Weather）和永久组合（Permanent Portfolio）的配置方案与预期收益。

### 2025 投资策略配置详情
[https://ricardojiang.github.io/ai-generate-code/harry/2025-config.html](https://ricardojiang.github.io/ai-generate-code/harry/2025-config.html)

详细展示 2025 年投资策略的具体配置参数和场景分析。

## 技术栈

- **前端框架**: React, Three.js
- **UI 库**: Tailwind CSS
- **图表库**: Chart.js
- **动画**: Anime.js
- **代码高亮**: Prism.js
